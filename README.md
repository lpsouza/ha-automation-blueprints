# Home Assistant Automation Blueprints

This repository contains a collection of Home Assistant automation blueprints created to enhance and simplify your smart home experience. Each blueprint is designed to be easily imported into your Home Assistant instance, allowing you to quickly set up automations without needing to write complex YAML code.

## Getting Started

To get started with these blueprints, follow these steps:

1. Browse the available blueprints in the repository.
2. Click on the blueprint you want to use.
3. Click the "Raw" button to view the raw YAML code.
4. Copy the YAML code.
5. In your Home Assistant instance, go to "Configuration" > "Automations" > "Blueprints" > "Import Blueprint".
6. Paste the YAML code into the editor and save.
